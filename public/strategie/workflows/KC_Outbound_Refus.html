KC_Outbound_Refus

Mini workflow — Clôture propre d'un refus pendant l'appel Sofia

🚫 DÉCLENCHÉ PAR SOFIA — ACTION "REFUS — FIN DE SÉQUENCE"
⚙️ Comment ce workflow est déclenché Ce workflow n'a pas de trigger GHL — il est lancé uniquement par Sofia (VoiceBot) via l'action Trigger a workflow configurée dans AI Agents → Sofia Outbound.
Sofia le déclenche quand le contact refuse clairement pendant l'appel.
📞 Déclenché par Sofia Quand le contact refuse clairement
"Trigger a workflow → KC_Outbound_Refus"
1
🏷 Add Tag KC_reactiv-no-response
+ → Add Tag Tag : KC_reactiv-no-response Marque le contact comme ayant refusé le service — visible dans le CRM pour les filtres et rapports de campagne.
2
🏷 Remove Tag KC_src-reactivation
+ → Remove Tag Tag : KC_src-reactivation Retire le contact de la liste de réactivation — il ne sera plus jamais rappelé si la campagne est relancée sur la même liste.
END
