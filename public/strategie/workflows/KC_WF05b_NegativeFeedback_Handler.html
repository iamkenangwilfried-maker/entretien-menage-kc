WF-05b — KC_NegativeFeedback_Handler

Traitement automatique des avis 1-3⭐ soumis via la page /demande-avis (Vercel)

⚡ TRIGGER : INBOUND WEBHOOK — PAGE /demande-avis → /api/feedback
✅ Prérequis déjà configurés • Webhook URL : https://services.leadconnectorhq.com/hooks/mF3A83KNLQxonw6GNCmY/webhook-trigger/535b0d0e-a000-409c-9206-80232b69ebdf
• Variable Vercel à ajouter : GHL_FEEDBACK_WEBHOOK_URL = URL ci-dessus → redéployer
• Custom fields créés : KC_rate_us (Number) · KC_your_feedback (Textarea)
• Mapping Reference configuré : inboundWebhookRequest.phone / .name / .rating / .comment
⚡ Inbound Webhook Page /demande-avis — client choisit 1, 2 ou 3 étoiles
POST → /api/feedback → GHL webhook
1
👤 Create / Update Contact Identifie le contact par téléphone
Mappe rating + commentaire
+ → Create/Update Contact Champs à mapper depuis le payload webhook : Phone → {{inboundWebhookRequest.phone}} First Name → {{inboundWebhookRequest.name}} KC_rate_us → {{inboundWebhookRequest.rating}} KC_your_feedback → {{inboundWebhookRequest.comment}} GHL retrouve le contact existant par téléphone et met à jour ses champs.
2

+ → Add Tag Tag : KC_avis-negatif Marque le contact comme ayant soumis un feedback négatif — utile pour filtres CRM, segmentation et suivi de résolution.
3
⛔ Remove from Workflow KC_ReviewRequest_Sequence (WF-05a)
+ → Remove from Workflow Workflow : KC_ReviewRequest_Sequence Stoppe immédiatement toutes les relances SMS en cours dans WF-05a — le client a répondu, plus aucun message ne part.

Notes importantes
Trigger unique (pas 3) : contrairement au GHL Survey natif, l'Inbound Webhook reçoit tous les ratings en un seul appel — pas besoin de 3 triggers séparés.
Identification du contact : GHL retrouve le contact via phone passé dans le payload. Le Trigger Link injecte ?phone={{contact.phone}} dans l'URL de la page.
Variable Vercel manquante : tant que GHL_FEEDBACK_WEBHOOK_URL n'est pas ajoutée dans Vercel, les feedbacks ne remontent pas dans GHL.
Avis positifs (4-5⭐) : ils ne déclenchent PAS ce workflow — le client est redirigé directement vers Google Review par la page /demande-avis.
