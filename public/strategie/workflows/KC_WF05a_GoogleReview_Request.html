WF-05a — KC_ReviewRequest_Sequence

Séquence 6 touchpoints avec Trigger Link gating — arrêt automatique dès le premier clic

⭐ TRIGGER : OPPORTUNITY STAGE → SERVICE COMPLÉTÉ

Prérequis — à créer dans GHL avant de construire ce workflow

1. Custom Contact Fields (Settings → Custom Fields → Contact) :
  — KC_rate_us (type : Number) | KC_your_feedback (type : Textarea)

2. Survey "KC_Review_Survey" (Sites → Surveys → + New Survey) :
  — Q1 : "Notez votre expérience" (Star Rating) → custom field : KC_rate_us
  — Q2 : "Votre commentaire" (Textarea) → custom field : KC_your_feedback

3. Trigger Link "KC_Review_Request" (Marketing → Trigger Links → + New) :
  — Action : Add Tag → KC_review-link-clicked
  — Redirect URL : URL de la page de notation KC (funnel avec étoiles 1-5)
  — Variable GHL : {{trigger_link.KC_Review_Request}}

4. Page de notation KC (Site KC ou GHL Funnel) :
  — Étoiles 4-5 → lien direct Google Review KC (s'ouvre automatiquement)
  — Étoiles 1-3 → affiche le Survey KC_Review_Survey

Notes importantes
Trigger Link : quand le client clique {{trigger_link.KC_Review_Request}}, GHL ajoute automatiquement le tag KC_review-link-clicked et redirige vers la page de notation. Chaque IF/ELSE détecte ce tag → arrêt immédiat des relances.
