ChampTypeRôleCapturé par

KC_Source_lead TEXT Widget / Formulaire / Appel / Réactivation / Référral — reporting ROI par canal Automatique via workflows
KC_Notes_service LARGE_TEXT Informations spéciales : accès, codes, préférences, allergies produits Manuel / Formulaire
KC_Statut_review TEXT Non demandé / Demandé / Reçu — suivi des avis Google par client WF-05 automatique
KC_Technicien_assigne TEXT Nom de l'équipe ou technicien assigné — communication interne Manuel après acceptation devis
KC_Zone_service TEXT Montérégie / Montréal / Autre — planification par zone géographique Automatique basé sur KC_Ville
