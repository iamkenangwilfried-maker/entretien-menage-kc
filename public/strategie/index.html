Workflows — 7 automatisations
✅ Système complet — 7/7 workflows construits. L'intégralité du parcours client est automatisé : capture du lead, relance des devis, réactivation outbound Sofia, demande d'avis Google et gestion des feedbacks négatifs.

Cliquez sur un workflow pour voir tous les détails, l'explication complète et le visuel de l'automatisation.

WF-01a
KC_Form_Inbound
Capture lead depuis le formulaire site web
Live
Déclencheur : Formulaire soumis — widget live call / site web
  • Crée / met à jour le contact GHL
  • Ajoute tags : KC_src-widget + KC_live-widget-lead
  • Place dans pipeline KC_Entretien → Nouveau Lead
  • SMS de confirmation au client (FR)
  • Email de confirmation au client
  • Email de notification interne → kcglobalservice.inc@gmail.com
WF-01b
KC_VoiceBot_Inbound
Traite les appels Jade — sauvegarde les infos collectées
Live
Déclencheur : Transcript Generated — quelques secondes après l'appel
  • Ajoute tags : KC_src-voicebot + KC_live-widget-lead
  • Place dans pipeline KC_Entretien → Nouveau Lead
  • Enregistre les champs KC : Ville, Type de service, Fréquence, Surface
  • SMS de confirmation + Email au client
  • Email de notification interne
WF-Devis
KC_DevisEnvoye_Relances
Séquence de relance automatique après envoi de devis
Live
Déclencheur : Étape pipeline → Devis Envoyé
  • J+1 : SMS de relance douce
  • J+3 : Email de suivi
  • J+7 : SMS d'urgence légère
  • J+14 : SMS final + Email d'alerte interne
  • Stop automatique si devis accepté
WF-05a
KC_ReviewRequest_Sequence
Sollicitation automatique des avis Google
Live
Déclencheur : Étape pipeline → Service Complété
  • 5 SMS en séquence avec Trigger Link vers /demande-avis
  • Stop automatique si le client clique le lien
  • 4-5⭐ → redirection directe vers Google Reviews
  • 1-3⭐ → formulaire feedback → WF-05b
WF-05b
KC_NegativeFeedback_Handler
Gestion des feedbacks négatifs — alerte immédiate
Publié
Déclencheur : Webhook entrant depuis /api/feedback (site web)
  • Crée / met à jour contact avec note + feedback texte
  • Ajoute tag KC_avis-negatif
  • Retire de la séquence WF-05a
  • Email d'alerte immédiat → équipe KC
  • Email de remerciement au client
WF-04b
KC_CallManagement
Gestion des non-réponses aux appels Sofia
Construit
Déclencheur : Call-End Workflow de Sofia (sur non-réponse)
  • Tentative 1 → attente 3h → Tentative 2
  • Tentative 2 → attente 24h → Tentative 3
  • Après 3 non-réponses → SMS de dernier recours au client
  • Tag KC_reactiv-retry-final → pipeline PERDU
  • Maximum 3 tentatives (anti-boucle infinie)
WF-04a
KC_Outbound_Caller
Lancement des appels sortants Sofia
Construit
Déclencheur : Tag ajouté → KC_src-reactivation
  • Envoie un appel IA via Sofia (KC_Outbound_Prospector)
  • Qualifié → Nouveau Lead dans pipeline
  • Pas intéressé → tag KC_reactiv-no-response
  • Pas de réponse → WF-04b (retry management)

Stratégie Marketing Social Media
Principe fondamental : Le service EST le contenu. 80% vidéo/photo terrain filmés par l'équipe KC, 20% visuels designés maximum. Ce qui fonctionne dans le nettoyage : les transformations avant/après, pas les flyers.

4 Piliers de Contenu

50%

Transformation

Vidéos avant/après, ASMR nettoyage, time-lapse. Format TikTok vertical 60–90s. Le plus viral.

25%

Preuve Sociale

Google Reviews, témoignages WhatsApp capture, réactions clients en temps réel.

15%

Ancrage Local

Tags Sorel-Tracy, Brossard, Longueuil. Groupes Facebook Montérégie. Références locales.

10%

Conversion

Plages disponibles, offres ponctuelles. Maximum 1 post de conversion par mois.

Plateformes Prioritaires

TikTok — Viralité

Format principal de production. Vidéos avant/après. Hashtags nettoyage en FR. Potentiel viral organique élevé.

Facebook — Leads 35+

Partage des TikToks dans les groupes locaux Montérégie. Facebook Ads pour génération de leads B2B et résidentiel.

Instagram — Repurpose

Republication automatique des TikToks. Reels + Stories. Vitrine visuelle du portefeuille KC.

Entonnoir Facebook Ads — Budget mensuel ~960$/mois

TOFU
Notoriété
5$/jour · Vidéo transformation · Audience froide · Objectif ThruPlay Cibler toute la Montérégie + Montréal. Vidéo avant/après la plus performante. Objectif : générer des vues 25%+ pour alimenter le MOFU.
MOFU
Intérêt
7$/jour · Témoignage + avant/après · Retargeting vidéo viewers 25%+ Personnes ayant regardé les vidéos TOFU. Message : preuve sociale + offre de soumission gratuite.
BOFU
Résidentiel
10$/jour · Formulaire Lead Meta → GHL · Tag KC_src-fb-ads Propriétaires Montérégie/Montréal. Formulaire pré-rempli → déclenche WF-02 GHL → soumission automatique.
BOFU
B2B
10$/jour · Gestionnaires d'immeubles · Cliniques · Entrepreneurs Ciblage par titre : Property Manager, Office Manager, Directeur des opérations, Médecin propriétaire.
Plan de démarrage : Mois 1 → TOFU seulement (5$/j). Semaine 3 → ajouter BOFU résidentiel (10$/j). Objectif : 25 leads/mois → 8-10 contrats signés. ROI cible : 960$ de budget → 8000$+ de revenus.
