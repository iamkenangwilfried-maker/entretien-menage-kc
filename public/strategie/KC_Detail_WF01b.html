Workflow 01b
KC_VoiceBot_Inbound
Se déclenche automatiquement quelques secondes après chaque appel Jade. Sauvegarde toutes les informations collectées pendant l'appel et notifie l'équipe KC.
✅ Statut : Live 🤖 Source : VoiceBot Jade 🎯 Pipeline : Nouveau Lead
⚡ Déclencheur
Type de trigger
Transcript Generated
Type = Calls · Direction = Inbound. GHL génère le transcript de l'appel quelques secondes après la fin de la conversation avec Jade. Ce trigger est indépendant des paramètres Post-Call de Jade — il se déclenche seul.
Pourquoi ce trigger et pas "Call-End Workflow" ? Le Call-End Workflow de Jade peut avoir un délai de plusieurs heures avant de se déclencher. Le trigger "Transcript Generated" se déclenche en quelques secondes. L'équipe reçoit les informations immédiatement après l'appel.
📋 Étapes du workflow
1

Enregistrer les champs KC collectés par Jade

Jade collecte pendant l'appel : la ville, le type de service, la fréquence, la superficie. Ces données sont sauvegardées dans les champs personnalisés KC du contact.

2

Ajouter les tags source

Ajout de KC_src-voicebot (origine : appel Jade) et KC_live-widget-lead (statut actif).

3

Créer une opportunité dans le pipeline

Opportunité créée dans KC_Entretien → Nouveau Lead. Le lead apparaît immédiatement dans le CRM avec toutes les infos du service.

4

SMS de confirmation au client

Le client reçoit un SMS lui confirmant que sa demande a bien été reçue et qu'un membre de l'équipe le contactera bientôt avec une soumission gratuite.

5

Email de confirmation au client

Email récapitulatif avec les détails du service demandé (type, ville, fréquence) et les prochaines étapes.

6

Email de notification interne

Alerte à kcglobalservice.inc@gmail.com avec le résumé complet de l'appel Jade : nom du client, service demandé, ville, notes supplémentaires.

🤖 Connexion avec Jade

Formulaire pré-appel (widget)

Avant de parler à Jade, le visiteur remplit : Prénom, Nom, Téléphone, Email. Ces données sont capturées automatiquement avant même que l'appel commence.

Pendant l'appel

Jade collecte : Ville, Type de service, Fréquence, Superficie, Nombre de pièces, Notes supplémentaires. WF-01b sauvegarde tout cela post-appel.
